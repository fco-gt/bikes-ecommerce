# Master Bikes
Pagina E-Commerce

## Tienda online de bicicletas.

## Tecnologias usadas:
- NextJS
- NextUI
- Embla
- Frame Motion
- React
- Tailwind CSS
- Tailwind Variants
- Next Themes
- Typescript/Tsx
- Javascript/Jsx

### Instrucciones de uso

Instalar dependencias

```bash
npm install
```

Ejecutar en modo desarrolador mediante

```bash
npm run dev
```
